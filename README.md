# cpp-ascii-raycast
Prototype raycast game using a console

Compiled using g++ (Debian 12.2.0-14) 12.2.0
g++ \*.cpp lib/\*.cpp -lncurses

Run the program from within a terminal.

Press 'q' to quit the program.
